)

This function converts UTF-16 string to UTF-32 string.

Parameters
[in]utf16_svconst reference to a string view representing UTF-16 string.
[out]utf32_sreference to a string which will hold converted string.
[in]comply_with_standardshould the conversion function comply with Unicode standard. Defaults to false. Refer to "Remarks" for details.
Returns
status specified by status_e enum. Judging by this Wikipedia article the standard does not allow usage of characters with code point from U+D800 to U+DFFF. Still, as noted by the aforementioned Wikipedia article you can convert "characters" in this range, though it is not recommended. By default the conversion is not strict. If you opt in to enable it you also must be able to handle status_e::non_standard_encoding return value separately.
